# Сортировка строк в Linux

Основной инструмент для сортировки строк в Linux: `sort`.

Команда читает строки из файла или стандартного ввода, сортирует их и печатает результат в стандартный вывод.

## Базовая сортировка

Сортировка строк файла по алфавиту:

```bash
sort names.txt
```

Сортировка вывода команды:

```bash
cat names.txt | sort
```

Практически удобнее передавать файл напрямую в `sort`, без `cat`:

```bash
sort names.txt
```

## Обратная сортировка

Для сортировки в обратном порядке используется `-r`:

```bash
sort -r names.txt
```

## Числовая сортировка

По умолчанию `sort` сравнивает строки как текст. Для чисел нужна опция `-n`:

```bash
sort -n numbers.txt
```

Пример:

```text
100
20
3
```

Обычный `sort` даст порядок `100`, `20`, `3`, а `sort -n` даст `3`, `20`, `100`.

## Игнорирование регистра

Чтобы не различать заглавные и строчные буквы, используется `-f`:

```bash
sort -f words.txt
```

## Удаление дублей

Если после сортировки нужно убрать повторяющиеся строки, используется `-u`:

```bash
sort -u names.txt
```

Альтернативный вариант через `uniq`:

```bash
sort names.txt | uniq
```

Важно: `uniq` работает корректно только с соседними одинаковыми строками, поэтому перед ним обычно делают `sort`.

## Сортировка по колонке

Если данные состоят из нескольких полей, можно сортировать по нужной колонке через `-k`.

Пример файла `users.txt`:

```text
alice 42
bob 7
carol 15
```

Сортировка по второй колонке как по числу:

```bash
sort -k2,2n users.txt
```

## Разделитель полей

Если поля разделены не пробелом, а, например, двоеточием, используется `-t`.

Пример:

```bash
sort -t: -k3,3n /etc/passwd
```

Это сортирует строки файла `/etc/passwd` по третьему полю.

## Практические примеры

Сортировка списка пакетов:

```bash
rpm -qa | sort
```

Сортировка списка файлов:

```bash
find . -type f | sort
```

Показать только первые 10 строк после сортировки:

```bash
sort names.txt | head
```

Показать последние 10 строк после сортировки:

```bash
sort names.txt | tail
```

Посчитать уникальные строки и отсортировать по количеству:

```bash
sort access.log | uniq -c | sort -nr
```

## Красивый вывод в колонки

Если после сортировки нужен более читаемый вывод, можно использовать `column`:

```bash
sort users.txt | column -t
```

Это особенно удобно для простых табличных данных.

## Полезные опции `sort`

- `-r` - обратный порядок.
- `-n` - числовая сортировка.
- `-u` - удалить дубли.
- `-f` - игнорировать регистр.
- `-k` - сортировка по колонке.
- `-t` - указать разделитель полей.

## Короткая памятка

```bash
sort file.txt
sort -r file.txt
sort -n file.txt
sort -u file.txt
sort -k2,2n file.txt
sort file.txt | uniq
sort file.txt | head
sort file.txt | tail
sort file.txt | column -t
```
