# Команда find в Linux

`find` используется для поиска файлов и директорий по разным условиям: имени, типу, размеру, времени изменения и другим параметрам.

Общий вид команды:

```bash
find <путь> <условия>
```

Пример:

```bash
find /var/log -name "*.log"
```

Эта команда ищет все файлы с расширением `.log` в `/var/log` и во вложенных каталогах.

## Базовое использование

Поиск всех объектов в текущем каталоге:

```bash
find .
```

Поиск в конкретном каталоге:

```bash
find /etc
```

## Поиск по имени

Точное имя:

```bash
find . -name "notes.txt"
```

Поиск без учета регистра:

```bash
find . -iname "readme.md"
```

Поиск по маске:

```bash
find . -name "*.conf"
find . -name "app-*"
```

## Поиск по типу

Только файлы:

```bash
find . -type f
```

Только директории:

```bash
find . -type d
```

Только символические ссылки:

```bash
find . -type l
```

## Поиск по размеру

Файлы больше 100 МБ:

```bash
find . -type f -size +100M
```

Файлы меньше 10 КБ:

```bash
find . -type f -size -10k
```

Файлы ровно 1 ГБ:

```bash
find . -type f -size 1G
```

## Поиск по времени изменения

Файлы, измененные за последние сутки:

```bash
find . -type f -mtime -1
```

Файлы, измененные более 7 дней назад:

```bash
find . -type f -mtime +7
```

Файлы, к которым обращались сегодня:

```bash
find . -type f -atime 0
```

## Ограничение глубины поиска

Искать только в текущем каталоге без вложенных:

```bash
find . -maxdepth 1
```

Искать начиная не с верхнего уровня:

```bash
find . -mindepth 2
```

## Комбинирование условий

Искать только `.log` файлы больше 10 МБ:

```bash
find /var/log -type f -name "*.log" -size +10M
```

Искать директории с именем `cache`:

```bash
find . -type d -name "cache"
```

## Выполнение команды для найденных файлов

С `-exec` можно запускать команду для каждого найденного файла.

Показать подробную информацию через `ls -l`:

```bash
find . -type f -name "*.sh" -exec ls -l {} \;
```

Найти и удалить временные файлы:

```bash
find . -type f -name "*.tmp" -exec rm {} \;
```

Более безопасный и быстрый вариант для удаления:

```bash
find . -type f -name "*.tmp" -delete
```

## Полезные практические примеры

Найти все файлы `Dockerfile`:

```bash
find . -type f -name "Dockerfile"
```

Найти все каталоги `.git`:

```bash
find . -type d -name ".git"
```

Найти большие логи:

```bash
find /var/log -type f -name "*.log" -size +50M
```

Найти файлы, измененные за последние 30 минут:

```bash
find . -type f -mmin -30
```

Найти пустые файлы:

```bash
find . -type f -empty
```

Найти пустые директории:

```bash
find . -type d -empty
```

## Полезные опции

- `-name` - поиск по имени с учетом регистра.
- `-iname` - поиск по имени без учета регистра.
- `-type` - фильтр по типу объекта.
- `-size` - фильтр по размеру.
- `-mtime` - поиск по времени изменения в днях.
- `-mmin` - поиск по времени изменения в минутах.
- `-maxdepth` - ограничить глубину поиска.
- `-mindepth` - начать поиск с указанной глубины.
- `-exec` - выполнить команду для найденного объекта.

## Короткая памятка

```bash
find .
find . -name "*.txt"
find . -type f
find . -type d
find . -type f -size +100M
find . -type f -mtime -1
find . -maxdepth 1 -type f
find . -type f -name "*.log" -exec ls -l {} \;
find . -type f -name "*.tmp" -delete
```
